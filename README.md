# Pizza Order Application Simulator

A class project developed for a Python programming course using Python and Tkinter. The application simulates a pizza ordering system with interactive GUI components, pricing logic, and receipt generation.

## Features
- Allows users to select pizza size, crust type, and toppings
- Calculates subtotal, tax, and total cost
- Displays an itemized receipt within the GUI

## Tools Used
- Python
- Tkinter

## Files
- Project 5 Pizza Application.py
- pizza_banner.gif

## How to Run
Run the Python file in VS Code or another Python IDE.
